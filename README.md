# Retail Sales Data Analysis (EDA)

## 📌 Objective
To analyze retail sales data and extract meaningful insights.

## 🛠 Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## 📊 Analysis Performed
- Data Cleaning
- Descriptive Statistics
- Time Series Analysis
- Product Analysis
- Customer Analysis
- Data Visualization

## 🔍 Key Insights
- Electronics category has highest sales
- Sales vary over time
- Few customers contribute most revenue

## 💡 Recommendations
- Focus on top-selling products
- Target high-value customers
- Improve marketing in low-sales periods

Retail Sales Dataset
